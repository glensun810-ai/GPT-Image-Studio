# DESIGN.md

## 气质与意象

清晨实验室——明亮、通透、精准。白色空间里一抹科技蓝划过，像极简建筑中的蓝色光带。干净利落的线条，克制的点缀，信息密度高但不压迫。属于专业工具的冷静美学，不是消费级产品的花哨。

## 配色方案

### 浅色主题（默认）
- **主背景**：近白（微蓝灰调），像清晨未落霜的实验室地面
- **卡片/面板**：纯白，带极轻阴影悬浮感
- **主强调色**：科技蓝 `#2563EB`（blue-600），用于按钮、选中态、关键交互
- **辅助强调**：浅蓝 `#3B82F6`（blue-500），用于 hover、次要按钮
- **成功色**：翠绿 `#10B981`（emerald-500），用于任务完成状态
- **警告色**：琥珀 `#F59E0B`（amber-500），用于进行中状态
- **危险色**：红 `#EF4444`（red-500），用于失败状态
- **文字**：主文字深灰近黑 `#0F172A`，次要文字中灰 `#64748B`

### 深色主题
- **主背景**：深墨蓝（oklch 0.14），沉稳不刺眼
- **卡片/面板**：微亮深蓝灰（oklch 0.19），层次分明
- **强调色**：同浅色主蓝，保持品牌一致性
- **文字**：浅灰白，对比度充足

## 界面尺寸

- **紧凑**（13px）：信息密度优先，适合大屏
- **默认**（15px）：平衡可读与密度
- **舒适**（17px）：可读性优先，适合小屏

## 字体排版

- 使用系统默认 sans-serif（PingFang SC / Microsoft YaHei），不做额外字体加载
- 标题：粗体，字间距紧凑
- 正文：常规字重，行高偏松（1.6+）

## 页面结构

- **顶部导航栏**：纯白底/深色底，底部细线分隔，左侧 Logo + 站名，右侧偏好设置 + API Key 入口
- **主区域左右分栏**：
  - 左侧（约 360px）：控制面板——prompt、参数、参考图、生成按钮
  - 右侧（弹性宽度）：结果展示区——任务状态 + 图片画廊
- **移动端**：单栏布局

## 动效与交互

- 生成按钮加载时微旋转图标 + 蓝色脉冲
- 任务进度条使用蓝色渐变
- 图片淡入 + 微缩放进入
- 所有过渡 200-300ms ease-out

## 设计禁忌

- 不要使用紫色渐变或彩色渐变
- 不要大面积使用纯黑文字，保持层次
- 不要过度装饰，保持工具产品的克制感
